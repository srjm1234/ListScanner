ListScanner Web路径扫描工具 - 功能模块分解
graph TD SCAN["ListScanner 扫描功能"] SCAN --> A["目标地址配置"] SCAN --> B["字典与扩展名管理"] SCAN --> C["生成候选路径"] SCAN --> D["多线程并发扫描"] SCAN --> E["扫描执行"] SCAN --> F["扫描报告生成"]
功能模块 子功能 说明
字典管理 选择字典 从 dictionary/ 目录中选择已有字典文件
编辑字典在线编辑字典内容，支持注释和 %EXT% 占位符
新建字典创建自定义字典文件
预览展开预览字典展开后的完整路径列表
扫描执行 多线程请求 使用 ThreadPoolExecutor 并发发送 HTTP GET 请求
风险分类调用 classifier.check_risk() 判断路径风险等级
进度跟踪实时更新完成数、发现数、错误数统计
结果展示 实时日志 终端风格实时显示扫描发现的路径
统计面板进度条 + 四项核心指标（总数/发现/高风险/错误）
结果表格展示状态码、风险等级、URL、大小等详细信息
风险筛选按高/中/低风险等级过滤结果
HTML报告扫描完成后自动生成独立的 HTML 报告文件
目标地址配置 URL 输入 支持带/不带协议前缀，自动补全 http:// 和尾部斜杠
扩展名配置 扩展名标签 点击切换启用/禁用，支持添加自定义扩展名
并发参数 线程数 / 超时 滑块调节线程数（1-50）和请求超时（1-30秒）
