说明：本系统采用 B/S（Browser/Server）架构。用户通过浏览器访问 WebUI（index.html）， 前端通过 REST API 与 Flask 后端通信。后端核心模块包括扫描器（WebScanner）、风险分类器（check_risk）、 字典处理器（read_wordlist）和报告生成器（save_html_report）。 数据资源包括配置文件（config.ini）和字典文件（dictionary/*.txt），扫描结果输出为独立 HTML 报告。
